# Assets Directory

Thư mục này chứa các file tài nguyên cho ứng dụng.

## Các file cần thiết (tùy chọn)

- `icon.png` - Icon ứng dụng (1024x1024px)
- `splash.png` - Splash screen (1242x2436px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px)

Nếu không có các file này, Expo sẽ sử dụng icon mặc định.

## Tạo assets nhanh

Bạn có thể tạo assets đơn giản bằng cách:
1. Sử dụng công cụ online như [App Icon Generator](https://www.appicon.co/)
2. Hoặc để Expo tự tạo bằng cách chạy `expo prebuild` (sẽ tạo assets mặc định)
