# Cách Build iOS trên GitHub Actions

## ✅ Build đã được trigger!

Workflow iOS đã được cập nhật và sẽ tự động chạy khi bạn push code.

## Xem trạng thái build

1. **Vào:** https://github.com/hieunh2xx/expense-manager/actions
2. **Chọn workflow:** "Build iOS"
3. **Xem build đang chạy** hoặc **nhấn "Run workflow"** để build thủ công

## Build thủ công (Nếu cần)

1. Vào: https://github.com/hieunh2xx/expense-manager/actions
2. Chọn "Build iOS" workflow
3. Nhấn "Run workflow" (góc phải trên)
4. Chọn branch: `master`
5. Nhấn "Run workflow"

## Download IPA sau khi build xong

1. Vào tab "Actions"
2. Chọn workflow run đã hoàn thành
3. Scroll xuống phần "Artifacts"
4. Download "ios-build"

## Thời gian build

- ⏱️ Build sẽ mất khoảng **10-20 phút**
- ✅ Bạn sẽ nhận được thông báo khi build xong

## Lưu ý

- Build chạy trên Mac runner miễn phí của GitHub
- IPA file sẽ được lưu trong Artifacts 30 ngày
- Sau khi download, dùng Sideloadly/AltStore để cài đặt
